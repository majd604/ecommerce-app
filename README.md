# Flutter E-Commerce App

A modern **E-Commerce Mobile Application** built with Flutter.

## Features

* User Authentication (Login / Register)
* Product Search
* Product Details
* Add to Cart
* Checkout System
* Stripe Payment Integration
* Order History
* Address Management
* Dark Mode Support
* - Email Verification
* - Password Reset
* - Google Sign-In

## Technologies

* Flutter
* GetX
* Firebase
* Stripe
* Cloudinary
